About

FA is a dream project of mine. I've always thought about making my own Minecraft clone, but I never understood 3D rendering or physics when I was 12, so started working with 2D and did all I could to make a Minicraft (A game by Notch, the creator of Minecraft, this game was made for the Ludum Dare Game Jam 2011). Since then, I have been clinging to the idea of releasing a top-down, Minecraft-like game. As I lost my old files to my old game, I remade it with better performance (switched from Turtle to Pygame), easy loading and saving, and I wanted to expand upon my idea with as many new updates and features as I could.

I know, this game looks like dog poo, but I'm intending to improve the game as time goes on, so please be patient. I'll listen to your comments and try to patch up the game as time goes on, with new updates, I'll add new features, new enemies, new bosses, new biomes, heck, even new dimensions!

It is nice to see people being curious about my game, don't worry, I'll keep up with your expectations.

Release Notes

Alpha-1 (Latest)

Alpha-1 was released on 19-Feb-2022 (A few bug fixes were released to the page earlier, so technically, the game was released earlier than the given date). It's the first-ever version of the game.

This version has, world loading and saving into a JSON file, screenshots, splash screen, debug (F3 equivalent feature from Minecraft), and 4 different biomes, with its own terrain generation quarks.

I promise I won't remove any of the features I implement in any updates starting from the beginning (Alpha-1).

Alpha-2 (In development)

This is a version of the game that I'm hoping would change some things about the world generation. Backwards compatibility with old worlds (Alpha-2) will be implemented ASAP with the release of this version. This conversion will most probably be done using a console application that comes with the game, an application that can convert any version of the game world to the latest version available. More features will be announced as I successfully implement them into the game.
